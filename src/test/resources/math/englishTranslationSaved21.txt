Так как любое борелевское множество является непрерывным образом множества иррациональных чисел, то $ { mathcal A} $ -множество может быть определено как непрерывный образ множества иррациональных чисел. Счетным пересечением и счетным объединением $ { mathcal A} $ -множеств является $ { mathcal A} $ -множество. Любой $ { mathcal A} $ -множество является измеримым по Лебегу. Свойство быть $ { mathcal A} $ -множеством инвариантно относительно борелевских измеримых отображений и $ { mathcal A} $ - операций (см. A-операция  $ { mathcal A} $ - операция ). Более того, для множества, которое является $ { mathcal A} $ -множеством, необходимо и достаточно, чтобы его можно было представить в результате операции $ { mathcal A} $, примененной к семейству замкнутых множеств. Существуют примеры $ { mathcal A} $ - множеств, которые не являются борелевскими; таким образом, в пространстве $ 2 ^ I $ всех замкнутых подмножеств единичного интервала $ I $ вещественных чисел множество всех замкнутых несчетных множеств является $ { mathcal A} $ -множеством, но не является борелевским. Любое несчетное $ { mathcal A} $ - множество топологически содержит совершенное канторово множество. Таким образом, $ { mathcal A} $ - устанавливает «реализацию» гипотезы континуума их мощность либо конечна, либо $  aleph_0 $, либо $ 2 ^ { aleph_0} $. Центр группы --:-- {{TEX | done}} Множество $ Z $ всех центральных элементов (иногда также называемых инвариантными элементами) группы, т. Е. Элементов, которые коммутируют со всеми элементами группы. Центр группы $ G $ является нормальной подгруппой и даже характеристической подгруппой в $ G $. Более того, каждая подгруппа центра нормальна в $ G $. Абелевы группы, и только они совпадают с их центрами. Группы, центры которых состоят только из единичного элемента, называются группами без центра или групп с тривиальным центром. Фактор-группа $ G / Z $ группы $ G $ по центру не обязательно является группой без центра. ==== Ссылки ==== [1] А. Г. Курош, «Теория групп», «1-2», «Челси» (1955-1956)
